Getting Started
Prerequisites

Node.js 18+

npm or yarn

Installation

Clone the repository:

git clone https://github.com/KevinZhu223/my-project-site.git
cd my-project-site


Install dependencies:

npm install
# or
yarn install


Start the development server:

npm run dev
# or
yarn dev


Open http://localhost:3000
 in your browser.
